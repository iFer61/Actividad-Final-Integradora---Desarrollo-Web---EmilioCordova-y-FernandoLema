## Proyecto final de Desarrollo Web - Tienda de Discos "Dj Vergüenza" 
Desarrollado por:
- Emilio Cordova
- Fernando Lema

## Descripción

Este es un sitio web estático para simular un e-commerce **orientado a la venta de discos de vinilo y CDs**, desarrollado como proyecto de aprendizaje de desarrollo web frontend. La web presenta un catálogo musical organizado por géneros, información de categorías y un formulario de contacto con validación de JavaScript.

### Temática
- Tienda de venta de discos músicales

### Páginas:
- **index.html** --> Inicio con sección de bienvenida y géneros destacados.
- **categorias.html** --> Vista detallada de los géneros musicales disponibles.
- **productos.html** --> Catálogo completo organizado por género.
- **contacto.html** --> Formulario de contacto con validación JavaScript.

### Estado del proyecto
Proyecto en desarrollo como ejercicio académico de frontend. Las imágenes de productos son obtenidas desde motores de busqueda en internet (Google Images, Bing Image Search), el formulario de contacto no está conectado a un backend real y cualquier información que se envíe no se almacenará.
